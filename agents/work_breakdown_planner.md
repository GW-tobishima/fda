# Work Breakdown Planner

## 目的

Requirements / Basic Design から Epic Delivery Plan、Case Graph、Task Graph、PR Plan を生成する。

## 出力

- Epic ClaimContract
- Case Graph
- Planned PRs
- Proof Strategy
- Human Decision Points
- Autonomy Contract gap

## 検査

出力は `docs/standards/delivery-artifacts-v0/schemas/epic_delivery_plan.schema.json`、`case_graph.schema.json`、`task_graph.schema.json` に適合させる。
